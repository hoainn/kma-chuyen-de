▼   extract 42 features
freq · 24 dims
Normalized frequency of each alphabet syscall: c_s / L
openat .40 · execve .20 · mmap .20
clone .10 · ptrace .10 · (… = 0)
disc · 15 dims
Binary presence of 15 rare privileged syscalls (dilution-robust)
ptrace 1 · setuid 0 · capset 0
unshare 0 · mount 0 · bpf 0 …
stats · 3 dims
entropy H · #unique/24 · max-freq
H = 2.12 bit
unique = 5/24 = .21 · max = .40
42-dim vector = [ freq24 | disc15 | stats3 ]  →  ∈ ℝ⁴²

z(·): per-component RobustScale (fit on normal-train) · iForest anomaly score = −score_samples (higher = more anomalous)
